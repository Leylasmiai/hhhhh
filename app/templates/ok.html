    In this page, we delve into the development of a web interface for our proposed BPELT (Bug Prediction with an Ensemble Learning Tool) approach. This interface facilitates the process of predicting software bugs by leveraging machine learning techniques to enhance prediction accuracy. Using datasets in the ARFF (Attribute-Relation File Format) format extracted from the PROMISE NASA repository, which is widely recognized in the field of bug prediction, our web interface allows users to upload datasets and view prediction results. It also provides visualizations in the form of charts and tables to help users comprehend the results and make informed decisions.
