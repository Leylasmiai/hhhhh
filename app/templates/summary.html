Description

The NASA Promise Repository is considered one of the most popular datasets used in the software bug prediction domain, and it includes Twelve several datasets such as KC1, KC3, JM1, CM1, MW1, PC1, PC2, PC3, PC4, PC5, MC1, and MC2 . Each dataset contains data between 22 to 40 features .The NASA dataset was developed under its Metrics Data Program, and in 2013, [Shepperd et al]. cleaned up duplicate and incompatible data from the dataset. This enhanced information can be found in the PROMISE database. Therefore, in our study, we used the NASA dataset for the purified data.The NASA data utilizes Halstead and McCabe metrics for each occurrence, and it includes approximately 40 features, such as unique operator value (MU1), unique operator value (MU2), the total number of operators (N1), the total number of operands (N2), and rows Code (LOC). These features are crucial in identifying software bugs and predicting their occurrence in the future.Overall, the NASA Promise Repository is an excellent dataset for evaluating the effectiveness of software bug prediction approaches. Its extensive coverage of features and publicly available data make it an ideal resource for researchers and practitioners in the field. By using this dataset, we can better understand software bugs and develop more effective approaches to prevent them.
